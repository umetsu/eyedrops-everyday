# eyedrops-everyday

緑内障患者向けの点眼管理アプリケーション

## 概要

eyedrops-everydayは、緑内障などにより毎日の点眼が必要な患者をサポートするFlutterアプリです。シンプルで使いやすいインターフェースで、点眼の記録管理、リマインダー機能、眼圧の履歴管理を提供します。

## 主な機能

- 📅 **点眼履歴カレンダー**: 月間カレンダーで点眼状況を視覚的に確認
- ⏰ **リマインダー通知**: 指定時刻での点眼通知
- 📊 **眼圧グラフ**: 眼圧の推移をグラフで表示
- 🎯 **シンプルUI**: 30代後半以降のユーザーに配慮した分かりやすいデザイン
