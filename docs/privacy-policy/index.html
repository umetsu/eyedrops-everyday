プライバシーポリシー

毎日目薬

最終更新日: 2026年6月27日

本プライバシーポリシー（以下「本ポリシー」）は、umeg.dev（以下「当方」）が提供するモバイルアプリケーション「毎日目薬」（以下「本アプリ」）における、利用者の情報の取り扱いについて定めるものです。

1. 基本方針

本アプリは、点眼の記録・リマインダー通知・眼圧の記録を、利用者ご自身の端末内で完結して行うアプリです。当方は、本アプリを通じて利用者の個人情報および利用データを一切収集しません。また、収集を行わないため、第三者への提供・共有も行いません。

2. 収集する情報

本アプリは、利用者から個人を特定できる情報（氏名、メールアドレス、電話番号、位置情報、端末識別子、広告 ID など）を一切収集しません。

利用者が本アプリに入力・記録する以下の情報は、すべて利用者の端末内にのみ保存され、外部のサーバーや当方、第三者に送信されることはありません。

  • 点眼の実施記録（日付、実施状況、実施時刻）
  • 眼圧の記録（測定日、測定値、左右の区別）
  • 通知時刻などのアプリ設定

これらのデータは端末内のローカルデータベースおよび端末内の設定領域に保存されます。

3. 外部送信・第三者提供について

本アプリは、データを端末外に送信する機能を持ちません。具体的には以下のとおりです。

  • インターネット通信を行いません。
  • アクセス解析ツール（Google Analytics 等）を使用していません。
  • クラッシュレポート収集ツールを使用していません。
  • 広告を表示せず、広告 SDK を使用していません。
  • 利用者のデータを第三者へ提供・販売・共有することはありません。

4. 端末の権限について

本アプリは、機能の提供のために以下の権限を使用します。いずれも利用者の情報を外部へ送信するためのものではありません。

  • 通知の表示: 点眼を促すリマインダー通知、および前日の点眼が記録されていない場合の通知を表示するために使用します。
  • アラームのスケジュール設定: 指定した時刻に正確に通知を行うために使用します。
  • バイブレーション: 通知時の振動のために使用します。
  • 端末起動時の処理: 端末の再起動後も通知のスケジュールを維持するために使用します。

5. データの保存と削除

本アプリのデータは、利用者の端末内にのみ保存されます。

  • アプリ内の操作により、記録の取り消し・修正ができます。
  • 本アプリをアンインストールすると、保存されていたすべてのデータは端末から削除されます。
  • データは端末内にのみ保存されるため、当方によるバックアップや復元は行われません。アプリの削除や機種変更を行うと、これまでの記録は失われますのでご注意ください。

6. お子様のプライバシー

本アプリは特定の年齢層を対象としていませんが、いかなる利用者からも個人情報を収集しません。

7. 本ポリシーの変更

当方は、必要に応じて本ポリシーを変更することがあります。重要な変更を行う場合は、本ページ上で改定後の内容を掲示します。変更後の本ポリシーは、本ページに掲載された時点から効力を生じます。
