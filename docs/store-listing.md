## 詳しい説明（Full description）

```
毎日の点眼、忘れていませんか？

「毎日目薬」は、緑内障などで毎日の点眼が欠かせない方のために作られた、シンプルでわかりやすい点眼管理アプリです。「今日はもう点眼したかな？」という不安をなくし、毎日の点眼を無理なく習慣づけられるようサポートします。

■ こんな方におすすめ
・毎日の点眼を続ける必要がある方
・点眼したかどうかをつい忘れてしまう方
・眼圧の経過を手軽に記録・確認したい方
・難しい操作が苦手で、シンプルなアプリを探している方

■ 主な機能

【ワンタップで点眼を記録】
ホーム画面のボタンを押すだけで、その日の点眼を「実施済み」として記録できます。間違えたときも、もう一度押すだけで取り消せます。面倒な入力はいりません。

【点眼カレンダー】
点眼の実施状況をカレンダーで一目で確認できます。点眼できた日が緑のマークで表示されるので、「昨日はちゃんと点眼したかな？」という振り返りや、点眼を忘れてしまった日の確認に役立ちます。過去の日付をタップして、後から記録を修正することもできます。

【リマインダー通知】
・定時通知：毎日決まった時刻に「点眼の時間です」とお知らせします（初期設定は21:00）。
・忘れ通知：前日の点眼が記録されていない場合、翌朝にお知らせします（初期設定は7:00）。
通知の時刻は、あなたの生活リズムに合わせて自由に変更できます。

【眼圧の記録とグラフ表示】
測定した眼圧（左目・右目）を記録し、折れ線グラフで経過を確認できます。1ヶ月・3ヶ月・6ヶ月・1年と表示期間を切り替えられるので、長期的な変化が把握しやすく、通院時の参考にもなります。

■ 安心して使えるプライバシー設計
・アカウント登録やログインは不要です。
・記録したデータはすべてお使いの端末内にのみ保存されます。
・インターネット通信を行わないため、データが外部に送信されることはありません。

毎日の点眼を、もっと確実に、もっと気軽に。
「毎日目薬」で、大切な目の健康習慣を続けましょう。

※ ご利用上の注意
・本アプリは「1種類の目薬を1日1回点眼する」方を対象としたシンプルな設計です。複数種類の目薬の管理や、1日に複数回の点眼の記録には対応していません。あらかじめご了承ください。
・データは端末内にのみ保存されるため、アプリを削除したり機種変更を行うと、これまでの記録は失われます。あらかじめご注意ください。
・本アプリは点眼の記録をサポートするものであり、医療上の診断・治療を行うものではありません。治療方針については医師の指示に従ってください。
```
